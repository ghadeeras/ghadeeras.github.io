Scalar Field Polygonizing

This is an example on polygonizing scalar field isosurfaces. A scalar field can be thought of as a function of which the domain is an N-dimensional space (3-dimensional space in this case), and the range is a set of scalar (real) values. An isosurface is a surface in the domain space of which every point is mapped to the same scalar value when applying the scalar field function.

To render such a surface, one needs to polygonize it or tessellate it first. One known algorithm for doing that is the marching cubes algorithm.

The algorithm used here is a variant of the marching cubes, called the marching tetrahedra.

For a deeper dive, see this blog post of Paul Bourke's.
