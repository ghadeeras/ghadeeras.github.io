Noise Toy

In this toy, I was just experimenting with generating random numbers inside the fragment shader. I eventually settled on using the xorshift algorithm.

Each pixel/fragment has a different RNG for which the seed is a mix of the fragment coordinates and a 128 bit uniform seed supplied by JavaScript. JavaScript initially generates a static buffer of 16K 32 bit seeds. For each frame it randomly copies 4 of these 32 bit seeds into the 128 bit uniform seed.

The fragment shader then uses the resulting seed to generate each pseudo random sample. It averages all the generated samples to output the final fragment color.
