Old MSX programs

These are some of my brother's and my oldest MSX Basic programs (from the late 80s / early 90s) that survived! :-)

These programs were written for an MSX computer system which was powered by a 16 bit Z-80 microprocessor. It had 64 KBytes of RAM, half of which was taken by the MSX Basic interpreter and other system routines.

In those days we used to save our programs on cassette tapes. After MSX emulators started to emerge, I decided to give these programs new life. I wrote an Object Pascal program back in 2001 that did a frequency demodulation to the signals stored on the cassette tapes in order to extract the binary data and save it in CAS files which MSX emulators accept as a cassette modality.
